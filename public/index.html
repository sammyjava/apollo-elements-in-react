Apollo Elements in React

How it Works

This demo uses Apollo Elements to query the unofficial SpaceX GraphQL API . The <spacex-launches> and <spacex-launch> custom elements are written with LitElement, but it could be any web component library, or none. Check out the docs for examples.

The <spacex-launches> element queries for a list of launches. When you select a launch from the dropdown, it fires a DOM event, which React listens out for to update it's own state. React then binds the selected launch's ID to the <spacex-launch> component with standard HTML and the DOM. <spacex-launch> then independently fetches details about the mission.
